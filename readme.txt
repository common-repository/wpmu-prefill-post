== Description ==
NOTE : This plugin is translation ready
NOTE : You need php 5.2+

Add the ability to create post template.
You can manage templates and use them to prefill a post.
Also prefill the post title.


Work with wordpress and wordpress mu.
Works with qtranslate.

Inspirated from "Article Templates"

== Installation ==
1. Unzip the php file and upload it into your blog
2. Activate the plugin in plugin's section
3. Now in "settings" menu you have a new submenu call "WPMU Prefill Post Template" where you can manage your templates.
